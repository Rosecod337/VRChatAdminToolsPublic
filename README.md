# VRChat Admin Tools [BETA]

Windows-приложение для администраторов и модераторов VRChat-сообществ. Оно помогает следить за текущим инстансом по локальным логам VRChat: входы и выходы игроков, смена аватаров, активность, общие заметки и быстрые отчёты для Discord.

> Discord: [discord.gg/7URkA6EQYY](https://discord.gg/7URkA6EQYY) - поддержка, обновления и сообщения об ошибках.

## Возможности

- Мониторинг текущего VRChat-лога в реальном времени.
- Отслеживание входов и выходов игроков с временем события.
- Отслеживание смены аватаров, если VRChat записал это событие в лог.
- Дашборд с текущей активностью и последними событиями.
- Admin Tools с общими метками и заметками по игрокам для вашей команды.
- Экспериментальный Crash Risk Analyzer для сохранения последних событий перед возможным сбоем VRChat.
- Быстрый Snapshot для копирования понятного отчёта.
- Analyze Server для восстановления недавних событий текущего инстанса.
- Необязательная проверка VRChat-аккаунта для более точного определения профиля и имён.
- Автообновления через GitHub Releases.

## Требования

- Windows 10 или Windows 11, 64-bit.
- Установленный VRChat, запущенный хотя бы один раз.
- Персональный лицензионный ключ.

## Установка

1. Откройте [Releases](../../releases/latest).
2. Скачайте `VRChat-Log-Analyzer-Setup-<version>.exe`.
3. Запустите установщик.
4. Откройте приложение и введите лицензионный ключ.
5. Нажмите **Start**, чтобы начать live-мониторинг.

## VRChat Cookie

VRChat account session cookie - необязательное поле. Оно может помочь приложению точнее определять профиль VRChat и имена игроков.

Используйте только свой cookie и не отправляйте его другим людям. Приложение хранит cookie локально на вашем компьютере.

## Analyze Server

**Analyze Server** полезен, если вы открыли приложение уже после входа в инстанс. Он анализирует недавние VRChat-логи и пытается восстановить события текущей сессии.

Для лучшей точности:

1. Нажмите **Start** до начала модерации.
2. Если возможно, перезайдите в нужный мир или инстанс.
3. Используйте live-мониторинг как основной источник данных.

Старые или неполные VRChat-логи могут пропускать события, поэтому анализ не гарантирует 100% точность.

## VR Overlay

Текущий VR Overlay - это прозрачное окно поверх рабочего стола. Чтобы видеть его внутри VR, можно захватить окно через Desktop+, OVR Toolkit, XSOverlay, Virtual Desktop или SteamVR desktop view.

## Приватность

- Приложение читает только лог-файлы, которые создаёт VRChat.
- Приложение не внедряется в VRChat, не меняет файлы игры и не взаимодействует с процессом игры.
- VRChat cookie необязателен и хранится локально.
- Проверка лицензии используется только для активации приложения и защиты доступа.

## Бета-статус

Проект находится в beta-версии. Некоторые счётчики могут быть неточными, если VRChat не записал нужные события в лог или если мониторинг был запущен уже после входа игроков.
