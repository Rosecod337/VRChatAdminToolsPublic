# VRChat Admin Tools [BETA]

Windows-приложение для администраторов и модераторов VRChat-сообществ. Оно помогает следить за текущим инстансом по локальным логам VRChat, видеть входы и выходы игроков, смены аватаров, активность сервера, заметки по игрокам и быстрые отчеты для команды.

## Сообщество

- Discord: [discord.gg/wXFuzxEbfC](https://discord.gg/wXFuzxEbfC)
- Telegram: [t.me/VRChatAdminTools](https://t.me/VRChatAdminTools)

## Возможности

- Live-мониторинг текущего VRChat-лога.
- Отслеживание входов и выходов игроков с временем события.
- Отслеживание смены аватаров, если VRChat записал событие в лог.
- Дашборд активности с текущим онлайном и графиком за последние часы.
- Analyze Server для восстановления недавних событий из текущего лога, последних логов или выбранных файлов.
- Admin Tools с общими метками и заметками по игрокам для вашей команды.
- Синхронизация заметок между администраторами одного серверного проекта.
- История сессий и краткие отчеты.
- Экспериментальный Crash Risk Analyzer: сохраняет события перед возможным сбоем и показывает только риск по совпадению времени.
- Автообновления через GitHub Releases.

## Что важно понимать

Приложение не является модом, читом или клиентской модификацией VRChat. Оно не читает память игры, не внедряется в процесс VRChat и не меняет файлы игры. Основной источник данных - локальные `output_log_*.txt`, которые создает сам VRChat.

Некоторые данные зависят от того, что VRChat успел записать в лог. Если приложение было открыто уже после входа игроков в мир, лучше нажать **Start** и перезайти в нужный инстанс, чтобы live-мониторинг собрал актуальные события.

## Синхронизация

Общие заметки и метки синхронизируются через сервер проекта. Если у администраторов отдельные ключи, но они используют один и тот же сервер проекта, они видят общую базу заметок.

## VRChat Cookie

VRChat account session cookie - необязательное поле. Cookie помогает приложению точнее проверять текущий VRChat-аккаунт, текущий инстанс и некоторые данные через VRChat API.

Как вставить cookie:

1. Установите расширение Cookie-Editor в браузер.
2. Откройте `vrchat.com` и войдите в свой VRChat-аккаунт.
3. В Cookie-Editor найдите cookie с именем `auth`.
4. Вставьте значение в приложение в формате `auth=authcookie_...`.

Cookie хранится локально на вашем компьютере. Приложение не пытается автоматически вытаскивать cookie из браузера.

## Analyze Server

**Analyze Server** полезен, если приложение открыли уже после входа в инстанс или нужно восстановить часть событий из логов.

В приложении можно выбрать:

- последние 3 лога;
- текущий лог;
- логи за сегодня;
- файлы вручную.

Перед анализом приложение спросит, насколько можно нагрузить ПК. Быстрый режим читает больше данных и может вызвать краткие лаги. Для стабильной работы лучше запускать **Start** заранее и перезаходить в мир, если нужна максимально актуальная картина.

## Установка

1. Откройте [Releases](../../releases/latest).
2. Скачайте `VRChat-Log-Analyzer-Setup-<version>.exe`.
3. Запустите установщик.
4. Откройте приложение и введите лицензионный ключ.
5. Нажмите **Start**, чтобы начать live-мониторинг.

## Бета-статус

Проект находится в beta-версии. Отдельные счетчики и восстановление старых событий могут быть неточными, если VRChat не записал нужные строки в лог или если мониторинг был запущен слишком поздно.

> Примечание: Сурс программы возможно откроется, когда прога выйдет из беты, чтобы у каждого была возможность сделать свой сервер и держать программу только для своих.
