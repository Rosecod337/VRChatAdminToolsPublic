# VRChat Admin Tools

Windows-приложение для администраторов и модераторов VRChat-сообществ. Оно анализирует локальные логи VRChat, показывает события текущего инстанса и помогает команде вести общие метки и заметки по игрокам.

Текущая стабильная версия: [`v1.0.2`](../../releases/latest).

## Сообщество

- Discord: [discord.gg/wXFuzxEbfC](https://discord.gg/wXFuzxEbfC)
- Telegram: [t.me/VRChatAdminTools](https://t.me/VRChatAdminTools)

## Возможности

- Мониторинг текущего VRChat-лога в реальном времени.
- Отслеживание входов, выходов и смены аватаров.
- Анализ текущего лога, логов за сегодня, последних логов или выбранных файлов.
- Дашборд онлайна, состояния приложения и последних событий.
- Admin Tools с общими метками, заметками и историей изменений.
- Crash Analyzer с несколькими возможными кандидатами и метками аватаров.
- Custom Builder с настраиваемыми блоками и компактным режимом.
- История игровых сессий и быстрые снимки состояния.
- Уведомления Windows и автоматические обновления через GitHub Releases.

## Что важно понимать

Приложение не является модом, читом или клиентской модификацией VRChat. Оно не читает память игры, не внедряется в процесс VRChat и не изменяет файлы игры. Основной источник данных — локальные `output_log_*.txt`, которые создаёт VRChat.

Некоторые сведения зависят от того, что VRChat успел записать в лог. Если приложение было запущено уже после входа игроков в мир, часть ранних событий может отсутствовать.

Crash Analyzer показывает совпадения по времени и событиям лога, но не доказывает вину игрока. Всегда проверяйте контекст перед модерационными действиями.

## Лицензионный ключ

Для запуска нужен персональный лицензионный ключ. Получить его можно через [Discord проекта](https://discord.gg/wXFuzxEbfC).

Не передавайте ключ другим людям: у каждого пользователя должен быть отдельный ключ. Если ключ не работает, истёк, заблокирован или достигнут лимит устройства, обратитесь в поддержку.

Опция **Запомнить меня** позволяет дольше сохранять сессию на текущем устройстве.

## Синхронизация

Общие заметки и метки синхронизируются между администраторами одной команды. У каждого администратора остаётся собственный ключ. Пользователи с одинаковым Team ID видят общие командные данные.

## VRChat Cookie

Cookie сессии VRChat — необязательное поле. Оно помогает точнее определить текущий аккаунт, инстанс и доступные через VRChat API сведения.

Как добавить cookie:

1. Установите Cookie-Editor в браузер.
2. Откройте `vrchat.com` и войдите в свой аккаунт.
3. Найдите cookie с именем `auth`.
4. Вставьте значение в приложение в формате `auth=authcookie_...`.

Cookie хранится локально на компьютере. Приложение не извлекает его из браузера автоматически.

## Анализ логов

В меню **Analyze Server** можно выбрать:

- текущий лог;
- логи за сегодня;
- последние три лога;
- файлы вручную.

Перед анализом приложение предложит выбрать допустимую нагрузку на компьютер. Быстрый режим обрабатывает больше данных одновременно и может вызвать кратковременные задержки. После анализа приложение автоматически продолжает читать новые события.

## Установка

1. Откройте [последний релиз](../../releases/latest).
2. Скачайте `VRChat-Log-Analyzer-Setup-1.0.2.exe`.
3. Запустите установщик.
4. Введите лицензионный ключ.
5. Нажмите **Start**, чтобы включить мониторинг.

Установщик пока не имеет платной цифровой подписи, поэтому Windows SmartScreen может показать предупреждение о неизвестном издателе. Загружайте приложение только из официального репозитория.
